# HR Schedule Hub
Приложение предназначено для:
- сбора индивидуального графика работы сотрудников;
- автоматизации процесса планирования и учета рабочего графика сотрудников.

## Возможности
### Авторизация:
- Авторизация сотрудников или руководителей.
---
### Роли пользователей:
- Сотрудник
- Руководитель
---
### Для сотрудников:
- Создание и планирование собственного рабочего графика;
- Отправка сформированного графика на подтверждение руководителю.
--- 
### Для руководителей:
- Просмотр графиков сотрудников;
- Подтверждение предложенного графика;
- Отклонение графика с возможностью обратной связи.
